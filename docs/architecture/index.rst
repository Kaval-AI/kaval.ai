Architecture & Concepts
=======================

Kaval.AI is designed to be a flexible and modular AI agent framework.

Overview
--------

The core components of Kaval.AI are:

* **Agents**: Responsible for task execution.
* **Workflows**: Defined in YAML, these specify the logic and flow of an agent.
* **Backoffice**: A management UI for configuring and monitoring agents.
